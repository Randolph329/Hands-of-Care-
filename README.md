Purpose:
Hands of Care is a senior-focused support app designed to restore connection,
choice, dignity, and safety for older adults.

Core Features:
- Senior Home Screen (large buttons, accessibility-first)
- Talk to a Live Person (with Emergency Help / 911 access)
- Things I Like (interests & identity)
- My Day (daily choices & freedom)
- Helper Services (background-checked, scheduled support)
